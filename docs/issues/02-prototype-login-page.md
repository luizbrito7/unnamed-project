# 02 - Página de Login

## Fase
Prototype

## Descricao
Criar a página de login (`/login`) com layout visual completo. Inclui:
- `LoginForm`: Formulário com campos email e senha, botão de login
- `ForgotPasswordLink`: Link para `/forgot-password`
- `RegisterLink`: Link para `/register`

Página pública (sem sidebar/header). Visual limpo e centralizado. Sem funcionalidade real — apenas UI estática.

## Criterios de aceite
- [ ] Página acessível na rota `/login`
- [ ] Formulário com campos de email e senha
- [ ] Botão de login com estilo primário
- [ ] Link "Esqueceu a senha?" apontando para `/forgot-password`
- [ ] Link "Criar conta" apontando para `/register`
- [ ] Layout centralizado, sem sidebar/header
- [ ] Responsivo para mobile
- [ ] Validação visual dos campos (required, tipo email)

## Dependencias
- Nenhuma

# 03 - Página de Cadastro

## Fase
Prototype

## Descricao
Criar a página de cadastro (`/register`) com layout visual completo. Inclui:
- `RegisterForm`: Formulário com nome completo, email, senha, confirmação de senha, cargo (opcional) e departamento (opcional)

Página pública (sem sidebar/header). Sem funcionalidade real.

## Criterios de aceite
- [ ] Página acessível na rota `/register`
- [ ] Formulário com campos: nome completo, email, senha, confirmação de senha
- [ ] Campos opcionais de cargo e departamento
- [ ] Botão de cadastro com estilo primário
- [ ] Link para voltar ao login
- [ ] Layout centralizado, sem sidebar/header
- [ ] Responsivo para mobile
- [ ] Validação visual (campos obrigatórios, tipo email, match de senha)

## Dependencias
- Nenhuma

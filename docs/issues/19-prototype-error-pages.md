# 19 - Páginas de Erro (404, 403)

## Fase
Prototype

## Descricao
Criar páginas de erro genéricas:
- **404 (Not Found)**: Página para rotas inexistentes com mensagem e link para dashboard
- **403 (Forbidden)**: Página de acesso negado com mensagem e link para dashboard

## Criterios de aceite
- [ ] Página 404 exibida para rotas inexistentes (Next.js not-found)
- [ ] Página 404 com mensagem amigável e botão para voltar ao dashboard
- [ ] Página 403 com mensagem de acesso negado e botão para voltar ao dashboard
- [ ] Visual consistente com o design system da aplicação
- [ ] Layout responsivo

## Dependencias
- Nenhuma

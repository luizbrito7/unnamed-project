# 04 - Páginas de Recuperação e Redefinição de Senha

## Fase
Prototype

## Descricao
Criar as páginas de recuperação de senha (`/forgot-password`) e redefinição de senha (`/reset-password`). Inclui:
- `ForgotPasswordForm`: Campo de email e botão para enviar link de recuperação
- `ResetPasswordForm`: Campos de nova senha e confirmação

Ambas são páginas públicas (sem sidebar/header). Sem funcionalidade real.

## Criterios de aceite
- [ ] Página `/forgot-password` acessível com formulário de email
- [ ] Botão de envio e mensagem de confirmação mockada
- [ ] Link para voltar ao login
- [ ] Página `/reset-password` acessível com campos de nova senha e confirmação
- [ ] Validação visual (email válido, senhas coincidindo, mínimo 8 caracteres)
- [ ] Layout centralizado, sem sidebar/header
- [ ] Responsivo para mobile

## Dependencias
- Nenhuma
